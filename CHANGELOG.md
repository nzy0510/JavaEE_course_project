# 更新日志

## 未发布

### 新增
- 知识库维护功能：支持知识条目的单条添加、批量 JSON 文件导入、编辑、归档和恢复。
- AI 智能问答：基于知识库内容的检索增强生成（RAG），调用 DeepSeek Chat API 生成回答。
- 用户认证：支持用户名+密码注册、登录和退出，密码 BCrypt 加密存储，Session 会话管理。
- 数据可视化：首页仪表盘展示知识条目统计卡片和分类饼图，统计页面展示分类分布、难度分布和状态占比。
- 预置种子数据：Spring、MySQL、Java 三大分类共 12 条知识条目，首次启动自动加载。
- 知识导入校验：JSON 批量导入时校验必填字段（标题、核心内容），非法条目跳过并提示错误。

### 变更
- 前端从 AI-Interview 项目的 Vue SPA 迁移为 JSP + Bootstrap 5 + jQuery，符合课程 JSP 架构要求。
- 数据库从 4 张知识库表简化为 1 张 `knowledge_atom` 表，移除向量搜索、版本管理、审查机制。
- 移除 Qdrant 向量数据库、Redis 缓存、Flyway 数据库迁移、邮件服务等依赖。
